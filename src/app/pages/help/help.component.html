Ajuda do Sistema

O que é o SCT?

O SCT (Sistema Coordenado de Transporte) é um sistema web que, se instalado, roda de modo offline sem a necessidade de internet. Todos os dados são armazenados localmente no dispositivo do usuário.

Quem pode usar? É pago?

Qualquer pessoa pode usar o SCT de acordo com o seu Termo de Uso. O sistema é gratuito e ninguém precisa pagar ou fazer nenhum tipo de contrapartida para usá-lo.
